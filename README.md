# ZIO Elasticsearch
ZIO Elasticsearch is a type-safe, testable and streaming-friendly ZIO native Elasticsearch client. 

Supported by following versions: 
- Scala: 2.12+
- ZIO: 2
- Elasticsearch: 7
- JVM 11+
